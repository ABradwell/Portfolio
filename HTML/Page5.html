You are able to add clickable-regions

Use the following pattern:

  • Add usemap = "#tag" in the <'img'> tag
  • Declare <'map' name="tag">
  • Within this tag, add the regions using <'area' shape = "circle/rect/poly/default" coords= "points" href = "link"
  • You can add many of these
